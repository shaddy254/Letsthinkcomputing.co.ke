The Central Processing Unit

image courtesy of trustedreviews.com

This is the main component of the computer where all execution and processing of computer programs takes place and also controls all other processes in the computer. Components of the CPU
  • Control Unit
  • Arithmetic Logic Unit
  • Registers

Control unit

Also referred to as the nerve center of the computer, this component supervises and oversees all the activities of a computer and monitors the execution of any program being processed.

Functions of the control unit

  • Coordinates and controls the activities of the computer.
  • Helps in retrieving and decoding instructions from the internal storage.
  • Controls activities of all other units connected to the computer by using the appropriate rule.
  • Communicates with I/O devices for transfer of data or results from storage.

Arithmetic Logic Unit

A part of CPU that is responsible for all the mathematical and logical calculations in a computer system. This part comprises of two parts ;
  • Arithmetic Logic Unit – this section performs all the arithmetic calculations including division, addition, subtraction, and multiplication.
  • Logic Section – performs all logic operation in a computer such as comparing, selecting, matching of data and sorting of data.

Registers

This unit stores instructions, data and intermediate results in a computer. It supplies information to other units of the computer when needed.

Functions of the cpu

Fetch - taking the instruction from the stored memories and transferring them to the Registers for execution. Decode – translating and decoding the assembly code. Store - after execution the results are stored temporarily on the memories. The CPU registers also store the instruction in execution and also those on their way to the secondary storage memories.

CPU sockets

Sockets are connectors on the motherboard that houses a CPU and forms an electrical interface and contact with the CPU. Some of the CPU sockets include;
  • Pin Grid Array (PGA) - this type of socket as the name suggests has some pins which are attached to the main carrier that houses the CPU and then the carier is attached to the motherboard.

  • Line/Land Grid Array - the LGA socket is the opposite of the PGA and hence it has the pins attached in the main sockets itself rather than on the motherboard.
