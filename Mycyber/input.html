INPUT DEVICES

Input devices are those devices that are connected to a computer and are used to transfer data into a computer,these devices include:Keyboard,scanners and mouse.

Keyboards

A keyboard is an input device that is used to enter data into a computer through pressing of keys present on the keyboard,any type of character or numeric data can be fed to the computer through the keyboard.

Types of keyboard keys

  • Typing Keys-Are the alphabetical letters from A-Z.
  • Numerical keys- these keys contain the the numerical numbers from 0-9.
  • Functional Keys- these keys contain the functional keys F1-F12 which are used as a combination with other keys to invoke certain processes or functions.
  • Navigation keys-are made up of the Arrow keys and the page up and page down keys which are used to navigate through the windows in a computer.

Mouse

Image courtesy of LOGITECHG.COM

A mouse is an input pointing device that is used to move a cursor pointer in the screen,a mouse generally detects a two dimensional motion relative to a surface after which the motion is transmitted to a pointer on a display screen .the first computer mouse was developed in 1963 by Douglas C. Engelbart.

other examples of Mouse include; Other examples of pointing devices

JOYSTICK

image courtesy of pinterest

A pointing input device that functions the same as a mouse only that it does not have any clickable button but one moves the cursor by just rotating a spherical ball that is located on the top of the device, this device can be rotated in all directions. Joysticks are mainly used Computer Aided Designing (CAD) or for computer gaming.

LIGHT PEN

image courtesy of madress

A pointing device that is similar to the normal pen but used select menus or draw pictures on the monitor or display screen, mainly found on PDAs and some tablets.

TRACK BALL

image courtesy of addesso.com

A device which is used to move the cursor by just moving /rotating a ball found on the upper side of the device while the other entire device is stagnant.

SCANNER

image courtesy of startups.co.uk

An input device used to capture data that is in human readable form and convert it into a computer readable form, the data that is recorded may then be stored in a computer or be formatted to form another document or information, scanners are of many types where we have optical and magnetic scanners.

MICROPHONE

image courtesy of currys.co.uk

An input device used to input sound into a computer or a multimedia device which is then stored in the computer, microphones are mainly to add sound or during communications using the computer.
